How It Works
SecretWatch monitors every text field and input box across all tabs.

When you press Enter to send, it scans for API keys, tokens, SSNs, credit cards, and emails before the message is sent. If secrets are found, you can Sanitize & Send (auto-redact) or Cancel to edit manually.
